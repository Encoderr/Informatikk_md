An internet service which connects domain names with IP-addresses to a service on the internet; which makes it possible to send the right information to the right place on the internet. 

- DNS is a **Distributed database** which is implemented in a hierarchy of many named servers.
- Uses Application-layer protocol, where hosts and the DNS communicate to resolve names.

- The main identifiers for the DNS is the SSN, name, passport etc..

- The IP address is used for addressing datagrams, and the "name" used by us humans can be the URL for example.
